2. Resumen (Abstract)

Este proyecto consiste en el diseño y desarrollo de un portfolio digital interactivo con una estética inspirada en el videojuego Flappy Bird. El desarrollo se ha centrado en la creación de una interfaz limpia y profesional utilizando exclusivamente HTML, CSS y JavaScript. Los objetivos principales alcanzados incluyen la implementación de un sistema multi-idioma dinámico, una estructura responsiva sin dependencia de recursos gráficos externos y una navegación intuitiva orientada a la presentación de habilidades técnicas en el sector tecnológico.

4. Introducción

En el actual mercado laboral tecnológico, la diferenciación es clave. Este proyecto nace de la necesidad de presentar una trayectoria académica de forma creativa. Se ha seleccionado la temática de 8-bits por gusto personal, sino como un reto de maquetación: construir una identidad visual potente basándose únicamente en código CSS. El objetivo específico es demostrar que el dominio de las hojas de estilo permite crear experiencias memorables sin necesidad de sobrecargar la web con imágenes pesadas.

Justificación

Se ha seleccionado la estética de 8 bits para demostrar que el diseño web puede ser altamente creativo y funcional utilizando únicamente herramientas nativas de maquetación.

5. Metodología

El proceso de trabajo siguió un flujo estructurado: primero, se realizó una fase de investigación analizando portfolios en plataformas como TikTok para identificar tendencias de micro-animaciones. Segundo, se definieron los colores y fuentes pixeladas. Tercero, se construyó la estructura semántica en HTML5. Finalmente, se programó la lógica en JavaScript para la gestión de idiomas (i18n) y la interactividad de los modales, utilizando herramientas como Visual Studio Code y siguiendo las enseñanzas de canales especializados como Kodeset.

6. Desarrollo o Implementación

La implementación técnica se apoya en tres pilares: 1. Flexbox y Grid para una simetría perfecta en las secciones de proyectos. 2. Variables CSS (:root) para gestionar una paleta de colores coherente.

Propiedades CSS Destacadas

Para lograr la simetría y el diseño sin imágenes, se implementaron las siguientes reglas:

.portfolio-grid { display: grid; grid-template-columns: repeat(2, 1fr); gap: 40px; }
:root { --fb-grass: #76b400; --fb-pipe: #73c32a; --font-pixel: 'Press Start 2P'; }

7. Resultados

Se ha obtenido una plataforma web funcional que integra: un CV completo, una carta de presentación formal y un portfolio de prácticas.

8. Discusión

Durante el desarrollo, el mayor desafío fue lograr que la estética de videojuego. Se decidió eliminar las imágenes de los proyectos para evitar distracciones visuales y la estructura de la información. La simetría de las columnas fue el punto más complejo.

9. Conclusiones

Este proyecto ha servido para consolidar conocimientos avanzados de CSS y la lógica de programación en JS.
